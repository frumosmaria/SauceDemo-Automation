SauceDemo Automation Tests

📌 Overview

This repository contains a comprehensive automation testing suite for SauceDemo
, a demo e-commerce website. The tests are designed to validate core functionalities including login, product ordering, cart management, and checkout workflow.

The suite is built for end-to-end (E2E) testing, ensuring reliability and stability of the application across different scenarios.

🛠️ Technologies & Tools

Programming Language: JavaScript / Python (adjust to your project)

Automation Framework: Cypress / Selenium WebDriver

Test Runner: Mocha / PyTest

Assertion Library: Chai / unittest

Version Control: Git & GitHub

CI/CD (Optional): GitHub Actions / Jenkins

⚡ Features Tested

User login with valid credentials

Adding products to the shopping cart

Viewing and validating the cart

Checkout process with personal details

Order confirmation message verification

Example scenario tested:

Positive Order Flow:

Login with standard_user

Add first product to the cart

Proceed to checkout

Fill in first name, last name, and zip code

Complete the order

Verify the order success message
